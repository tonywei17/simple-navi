# 产品与设计指南 - 简单导航 (Simple Navi)

## 1. 项目愿景 (Vision)
简单导航专为**老年用户 (60+)** 设计，旨在消除传统地图导航的认知障碍。其核心理念是：“不论老人在哪，箭头永远指着家”。

## 2. 目标用户 (Target Audience)
*   **核心用户**：对复杂地图、图标和过载信息感到困惑的老年人。
*   **家庭成员**：协助设置初始地址并教老人如何简单阅读箭头的家属。

## 3. 核心功能 (Core Features)
*   **极简导航**：全屏大号箭头指向目的地，辅以超大字体显示的直线距离。
*   **地址三槽位**：支持存储最多3个常用地址（家、工作、其他/子女家）。
*   **地图微调**：允许通过地图交互精确校准目的地坐标，应对模糊地址输入。
*   **动态指引**：支持 iOS 灵动岛 (Dynamic Island) 与实时活动 (Live Activities) 持续指引。

## 4. UI/UX 设计规范 (Elderly-Friendly Design)
### 4.1 视觉原则
*   **低认知负载**：去除背景地图干扰，仅保留方向感。
*   **增强可读性**：48pt 粗体数字，高对比度色彩（蓝/紫指引，橙/红提醒）。
*   **动态适配**：当距离数字过大时，布局自动由水平切换为垂直，确保不截断。

### 4.2 交互模式
*   **触觉反馈**：按钮点击伴有物理震动感 (ScaleEffect)。
*   **单一焦点**：每个界面只强调一个主要操作（如“保存”或“切换目的地”）。
*   **错误宽容**：当未设置地址时，通过直观的弹窗引导设置，而非报错。

## 5. 辅助功能 (Accessibility)
*   完全适配 **VoiceOver**，所有交互组件均带语义化标签。
*   支持系统 **动态字体 (Dynamic Type)** 缩放。
*   所有核心图标 (SF Symbols) 均使用 `.semibold` 或 `.bold` 样式，增加边缘清晰度。
