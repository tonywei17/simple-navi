# 开发日志 2026-03-04

## 概要

本次更新包含三大改动：**适老化 UI 重设计**、**iPad 布局修复**、**中文品牌名更新与本地化修复**。

---

## 完成任务

### 1. 适老化 UI 重设计（琥珀金 + 钢蓝配色）

- 创建 `DesignTokens.swift` 集中管理设计令牌（颜色、透明度、字号最低值）
- 配色方案：琥珀金（#C29954）主色调 + 钢蓝（#337FB3）箭头 + 暖铜色（#C76B33）捐赠页
- 毛玻璃效果从 `.ultraThinMaterial` 改为 `.thinMaterial`，提升可读性
- 卡片边框改为琥珀金描边（`.accent.opacity(0.25)`）
- 所有 `.secondary` 文本色替换为自适应明暗模式的 `DesignTokens.textSecondary`
- 最小可读字号设为 18pt，状态文本设为 20pt
- 阴影透明度从 0.05 提升至 0.10

**涉及文件：**
- `DesignTokens.swift`（新建）
- `GlassCardModifier.swift`（新建）
- `LayoutMetrics.swift`（新建）
- `CompassView.swift` — 背景渐变、罗盘环、刻度、方向标签、距离文本、顶栏按钮
- `CompassArrow.swift` — SF Symbol 渐变改为钢蓝色
- `SetupView.swift` — 背景、按钮、验证文本
- `AddressMapConfirmView.swift` — 错误提示 14pt→18pt bold、确认按钮
- `DonationView.swift` — 暖铜色调、头部间距修复
- `LanguageSelectionView.swift` — 琥珀金选中状态

### 2. iPad 布局一致性修复

- 距离卡片添加 `.padding(.horizontal, metrics.horizontalMargin)`
- 地图预览卡片从硬编码 `16` 改为 `metrics.horizontalMargin`
- 目的地图标切换器横屏模式 padding 从 `metrics.cardPadding` 改为 `metrics.horizontalMargin`

### 3. 中文品牌名更新

- APP 内中文名称从「极简导航」改为「丢了吗」
- 副标题改为「极简导航 · 设置重要地址」
- 英文和日文名称保持不变

### 4. APP 名称本地化修复

- **问题**：中文系统下 APP 名称仍显示 "Simple Navigation"
- **根因**：
  1. Build Settings 中 `INFOPLIST_KEY_CFBundleDisplayName = "Simple Navigation"` 会覆盖本地化文件
  2. `InfoPlist.strings` 未注册到 Xcode 项目（project.pbxproj）
- **修复**：
  1. 删除 Debug/Release 中的 `INFOPLIST_KEY_CFBundleDisplayName`
  2. 添加 `PBXVariantGroup` 将 en/zh-Hans/ja 三个 InfoPlist.strings 注册到项目资源

## 技术决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 配色方案 | 琥珀金 + 钢蓝 | 暖色调亲和力强，钢蓝方向箭头醒目，适合老年用户 |
| 设计令牌管理 | 集中式 enum DesignTokens | 所有视图引用统一常量，避免硬编码分散 |
| 毛玻璃材质 | thinMaterial | 比 ultraThinMaterial 不透明度更高，文字更清晰 |
| 最小字号 | 18pt | 老年用户可读性底线，错误信息加粗处理 |

### 5. iPad 卡片宽度一致性修复

- **问题**：iPad 自适应布局中，距离卡片和目的地卡片宽度可能不一致
- **修复**：在两张卡片的 `.padding(metrics.cardPadding)` 之后、`.glassCard()` 之前添加 `.frame(maxWidth: .infinity)`，确保两张卡片始终撑满可用宽度
- **涉及文件**：`CompassView.swift`（destinationCard + distanceCard）

### 6. Xcode Scheme 修复

- **问题**：Widget scheme 的 `_XCWidgetKind` 环境变量为空，导致调试报错；主 App scheme 仅存在于旧用户目录下，当前用户不可见
- **修复**：
  1. 设置 `_XCWidgetKind = "SimpleNaviWidgets"`
  2. 将 SimpleNavi 主 App scheme 复制到 `xcshareddata/xcschemes/` 共享目录
